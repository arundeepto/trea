🔧 প্রথমবার সেটআপ দরকার

এই অ্যাপটা চালাতে নিজের একটা ফায়ারবেস প্রজেক্ট লাগবে। ধাপগুলো:

  1. Firebase Console-এ গিয়ে একটা প্রজেক্ট বানাও।
  2. Build → Authentication → Sign-in method → Email/Password চালু করো।
  3. Build → Firestore Database → একটা ডাটাবেস তৈরি করো (production mode)।
  4. Firestore এর Rules ট্যাবে গিয়ে নিচেরটা বসাও (শুধু লগইন করা ট্রেজারারই লিখতে/পড়তে পারবে): 
    rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
  5. Project settings → General → নিচের দিকে "Your apps" → Web app যোগ করে firebaseConfig অবজেক্টটা কপি করো।
  6. এই ফাইলের <script type="module"> এর শুরুতে firebaseConfig এর জায়গায় নিজেরটা বসিয়ে ফাইলটা আবার সেভ/আপলোড করো।

এই ধাপগুলো একবারই করতে হবে। এরপর থেকে যে কেউ ট্রেজারার হিসেবে অ্যাকাউন্ট বানিয়ে লগইন করতে পারবে এবং সব হিসাব ফায়ারবেসে জমা থাকবে।
